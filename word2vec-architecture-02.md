### ハッシュテーブルによる文章の表現

| 辞書  | 単語ハッシュ値 |
| ----- |:--------------:|
| a     | 1              |
| is    | 2              |
| pen   | 3              |
| this  | 4              |

| 文章 |      |      |      |
|:----:|:----:|:----:|:----:|
| this | is   | a    | pen  |
| 4    | 2    | 1    | 3    |
